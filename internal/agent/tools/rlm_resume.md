Query for session resumption context - answers "What was I working on?"

This tool retrieves context from previous sessions to help you resume work effectively. It returns:

- **Summary**: What was accomplished in the last session
- **Unfinished Work**: What wasn't completed
- **Next Steps**: Recommended starting points
- **Active Files**: Files that were being worked on
- **Key Insights**: What was learned
- **Blockers**: Issues that were encountered

Use this tool at the start of a session when:
- The user asks "What was I working on?"
- You need context about previous work
- You want to understand where to continue

Parameters:
- `include_recent` (optional): Include summaries from multiple recent sessions

The tool returns structured context that helps you:
1. Understand the current state of work
2. Identify what needs attention
3. Resume effectively without losing context
